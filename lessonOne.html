Lesson 1

In this lesson, we will cover the basics of HTML and CSS. How can they be combined to make the front end of the website?

What is HTML?

HTML stands for HyperText Markup Language. It is used to create a website. Combined with tags and elements, HTML is used for almost every website on the internet and generally easy to use.

What is an Element?

An element is a feature of HTML that can contain content. It is generally surrounded by opening tags < and end tag >. There are many elements included semantic (have meaning) and non-semantic elements.

Non-Semantic Elements: div, span, br

Semantic Elements: nav, article, header

Basic Structure of HTML

Generally, you would first set the language of the HTML document. Then, you would create a head tag which includes the meta-data, title, and any links to CSS or JavaScript files. Then, the body tag can be created which includes every element of the website. Usually, by pressing "!", you are able to automatically create the base code.

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style HTML elements and add visuals, color, and designs. It is mostly for appeals and aesthetics but can also create animations and viewports

What is a Selector?

A selector allows the user to choose an element and style it. To choose an element, you would usually write the element name followed by open and closed brackets. All the designs would go in there.

Inline vs Block Elements

Inline elements are elements that are placed inside of another element. They can share the same line as the other element. Block elements are elements that are placed on a new line.

Inline elements: span, a, img, br

Block elements: p, div, h1, form
