In this lesson, we will cover the basics of Javascript. How can Javascript be used to add more functionality to our websites.

What is Javascript?

Javascript is a programming language that is used to add more functionality to websites. Typically, the syntax and format is similar to Java which functions, variables, parameters etc.

Connecting HTML and Javascript

To connect HTML and Javascript, we need the <link> and then you can link the file name or path to your script.

You can put this script on the head so it will asynchronously load and run before the body loads. Or you can put it on the body and it will load after the body loads.

What is a Variable

A variable is a container that stores information. It can be a number, string, boolean, object, array, etc. To create a variable, you can just type the variable name followed by a equal sign to the value you want to create. But there different ways you can create variables:

var: can be used to create a variable globally

let: can be used to create a variable locally

const: can be used to create a variable that cannot be changed

Strings, Numbers, and Booleans

Strings are used to store text. Typically, they are denoted with parenthesis. They are surrounded by quotation marks. Numbers are used to store numbers. Booleans are used to store true or false values.

What is a Conditional

A conditional is a statement that is used to check if a certain condition is true.

The syntax for a conditional is: if (conditional statement) {code here}

Conditional operators include: <, >, ≤, ≥

What is an Iteration

An iteration is a loop that is used to repeat a certain code a certain amount of times

The syntax for an for-loop is: for (let i = 0; i < 10; i++) {code here}

The syntax for a while-loop is: while (conditional statement) {code here}
